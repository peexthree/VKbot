# AGENTS.md — Инструкции для Jules (и всех будущих агентов)

**Проект:** VK Esoteric Bot — «MindSee» / «Призрачный Интерфейс»

## 1. Общая идеология проекта

Бот — это **полноценный SaaS-продукт уровня крупной компании** в нише эзотерики (таро, астрология, магия).  
Главная цель — превратить общение с ботом в ощущение **мобильного приложения** внутри VK.

**Ключевая концепция:**
- **«Призрачный интерфейс»** — максимальное использование `messages.edit()` + inline callback. Минимум новых сообщений, никакого чат-спама.
- **«Живой ритуал»** — все длительные операции сопровождаются `start_dynamic_typing()` с театральными фразами каждые 10 секунд.
- **«Путь Адепта»** — чёткая онбординг-воронка: регистрация → первая карта дня → гримуар → глубокие платные разборы.

## 2. Архитектура и структура

- `main.py` — точка входа, warmup_task, Sentry, глобальный error handling
- `modules/` — все обработчики (registration, profile, services, tarot, payments, admin)
- `configs/` — модели Gemini, константы
- `prompts/` — все системные промпты и персоны (personas.py, base.py)
- `cards/` — изображения карт и ассеты
- `templates/` — Jinja2 для PDF
- `database.py` — работа с Supabase
- `ai_service.py` — вся логика Gemini + skins + фильтры
- `.jules/` — технические заметки (bolt.md, sentinel.md)

Глобальный `bot` импортируется только из `modules.bot_init`.

## 3. UX / UI Правила (строго соблюдать)

- Все навигационные кнопки (Профиль, Услуги, Гримуар) — **edit текущего сообщения**.
- Перед любой тяжёлой операцией: `callback_answer` + очистка клавиатуры (`edit` с пустой inline клавиатурой).
- Dynamic typing (`start_dynamic_typing` / `stop_dynamic_typing`) обязателен для всех AI-запросов и генераций.
- Форматирование текста: **только тире «-»**, CAPS для акцента, **никакого Markdown**.
- Персона — премиум-таролог, тёплый, эзотерический стиль (Instagram-level).

## 4. Ключевые механики

- **Гримуар** — система коллекционирования. 22 Старших Аркана → автоматический unlock скина «Магистр».
- **Карта Дня** — бесплатная, 24ч cooldown + страйки (7 дней = бесплатный синтез недели).
- **Синастрия и Оракул** — сбор данных **до** списания энергии + подтверждение.
- **Экономика** — «Энергия звёзд» (10 = 1 RUB). Списание только после успешной генерации.
- **Рефералка и Синдикат** — динамический прогресс, ветераны (старше 24ч) не могут вводить коды.

## 5. Технические стандарты и правила (обязательно)

- **Concurrency:** Redis locks (`acquire_lock` / `release_lock` в try…finally). Не re-entrant.
- **Error Handling:** try…finally + cleanup dynamic_typing. При ошибке в платном флоу — «Технические работы, энергия не списана».
- **Sentry:** инициализирован в main.py с SENTRY_DSN.
- **Warmup:** async warmup_task в main.py — preload cards/ с Redis lock и рандомными паузами.
- **AI Safety:** pre-filter 18+ по stop-words перед запросом к Gemini.
- **PDF:** только по отдельной кнопке «СГЕНЕРИРОВАТЬ PDF».
- **CI:** Ruff + Mypy + pytest (pyproject.toml). Локально: `PYTHONPATH=. pytest -v --asyncio-mode=auto`
- **Render:** AST monkey-patch в начале main.py для Python 3.14+.
