**VK Esoteric Bot** — это высокотехнологичная SaaS-платформа для социальной сети ВКонтакте, предоставляющая персонализированные эзотерические услуги на базе искусственного интеллекта (Google Gemini 2.0).

Бот спроектирован как полноценное мобильное приложение внутри VK с использованием «призрачного интерфейса» (максимальное редактирование сообщений вместо спама) и глубокой игровой механикой.

---

## 🚀 Преимущества как SaaS-решения

Продукт готов к коммерческому запуску «из коробки»:

1.  **Готовая монетизация:** Полная интеграция с **VK Pay** и **VK Donut**. Внутренняя валюта «Энергия звёзд» (курс 10 ✨ = 1 ₽).
2.  **Анти-фрод система:** Списание энергии происходит только после успешной генерации ответа ИИ.
3.  **Админ-панель:** Встроенная консоль управления пользователями, балансами, рассылками и аналитикой прямо в интерфейсе бота.
4.  **Реферальная система «Синдикат»:** Многоуровневая система приглашений с защитой от накруток (24ч окно активации).
5.  **Масштабируемая архитектура:** Поддержка Webhooks для высокой нагрузки, Upstash Redis для кэширования и Supabase как надежная БД.
6.  **ИИ-Скины:** Система персонажей с уникальными стилями общения (Олеся, Шэпс, Распутин и др.).

## ✨ Основные возможности

*   🃏 **Карта Дня:** Бесплатный ежедневный ритуал с накопительными бонусами (стрики).

## 🛠 Быстрый старт

### 1. Локальный запуск
1.  Клонируйте репозиторий.
2.  Установите зависимости:
    ```bash
    pip install -r requirements.txt.old  # Или используйте poetry
    ```
3.  Скопируйте `.env.example` в `.env` и заполните переменные (см. ниже).
4.  Запустите бота:
    ```bash
    python main.py
    ```

### 2. Деплой на Render
1.  Создайте новый **Web Service**.
2.  Укажите Runtime: `Python 3.11+`.
3.  Build Command: `pip install -r requirements.txt.old`.
4.  Start Command: `python main.py`.
5.  Добавьте все переменные из `.env` в разделе Environment Variables.
6.  Порт по умолчанию: `10000`.

## 🔑 Переменные окружения (.env)

| Переменная | Описание | Где получить |
| :--- | :--- | :--- |
| `VK_TOKEN` | Токен сообщества VK с правами доступа к сообщениям. | Настройки сообщества -> Работа с API |
| `GEMINI_API_KEYS` | Список ключей Gemini через запятую (для ротации). | [Google AI Studio](https://aistudio.google.com/) |
| `SUPABASE_URL` | URL вашего проекта Supabase. | [Supabase Console](https://supabase.com/) |
| `SUPABASE_KEY` | Сервисный ключ (anon или service_role). | Supabase -> Settings -> API |
| `UPSTASH_REDIS_URL`| URL Redis (rediss://...). | [Upstash](https://upstash.com/) |
| `ADMIN_ID` | Ваш цифровой ID ВКонтакте. | [Узнать ID](https://vk.com/link) |
| `VK_CONFIRMATION_CODE`| Код подтверждения сервера (для Webhook). | Настройки сообщества -> Callback API |
| `VK_SECRET_KEY` | Секретный ключ Callback API. | Настройки сообщества -> Callback API |

> **Важно:** Для корректной работы БД выполните SQL-запрос из файла `database_sync.sql` в редакторе Supabase.

## 🏗 Структура проекта

*   `main.py` — Точка входа, веб-сервер, воркеры и расписание (cron).
*   `ai_service.py` — Ядро интеграции с Gemini ИИ.
*   `modules/` — Функциональные модули:
    *   `admin.py` — Консоль управления.
    *   `tarot/` — Логика раскладов и Оракула.
    *   `payments/` — Обработка платежей и VK Pay.
    *   `registration/` — Онбординг и сбор данных.
*   `prompts/` — Системные инструкции и настройки персонажей.
*   `templates/` — HTML-шаблоны для генерации PDF.
*   `database/` — Слой работы с Supabase.

## 📖 Для разработчиков

### Как добавить новый скин?
Добавьте описание персонажа в `prompts/personas.py` и ссылку на изображение в `modules/utils/consts.py` (словарь `SKIN_ASSETS`).

### Как добавить новый расклад?
1.  Опишите логику в `modules/tarot/`.
2.  Добавьте кнопку в `modules/keyboards.py`.
3.  Зарегистрируйте обработчик в `modules/services.py`.

### Тестирование
```bash
PYTHONPATH=. python -m pytest tests/
```

## ❗ Troubleshooting

*   **VKAPIError_9 (Flood Control):** Бот автоматически делает паузы между запросами. Если ошибка сохраняется, увеличьте задержку в `modules/utils/ui.py`.
*   **Ошибка 911 (Keyboard invalid):** VK ограничивает клавиатуры 6 рядами по 5 кнопок. Проверьте динамическую генерацию.
*   **Пустой PDF:** Убедитесь, что установлены системные зависимости для WeasyPrint (особенно на Linux).
