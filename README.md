# VK Esoteric Bot — Таро • Астрология • Магия

**Продвинутый VK-бот с ИИ (Gemini)** для персонализированных эзотерических консультаций.
Уровень продукта — готовая SaaS-платформа.

## ✨ Основные возможности

- **Ежедневные персональные транзиты** — автоматически в 12:00 UTC
- **Расклады Таро** (карта дня, полные расклады, PDF-отчёты)
- **Несколько скинов** (Олеся, Шэпс, Распутин и др.) с уникальным стилем общения
- **Глубокая персонализация** (имя, дата рождения, теги, синестрия)
- **Платежи и подписки** (VK Pay / Donut) + защита от фрода
- **Админ-панель** и система рефералок
- **Генерация PDF** с картами и красивыми шаблонами

## 🚀 Деплой (Render / любой VPS)

1. Создай Web Service
2. Укажи команду запуска: `python main.py`
3. Добавь все переменные окружения (см. `.env.example`)
4. Порт: `10000`

## 📋 Переменные окружения

Смотри файл `.env.example` в репозитории.

## 🏗 Технологии

- **Python 3.11+**
- **vkbottle 4.x** (polling)
- **Gemini 2.0** (Flash + Pro + fallback)
- **Supabase** (база пользователей)
- **Upstash Redis** (кэш + FSM)
- **Pillow + WeasyPrint + Jinja2** (PDF)
- **loguru** + ротация логов
